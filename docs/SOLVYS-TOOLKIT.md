## Technical Specifications

### Pitch Deck Landing Page Requirements
- **Framework**: React + TypeScript (or vanilla HTML/CSS/JS)
- **Animations**: Smooth scroll-triggered (GSAP or Framer Motion recommended)
- **Performance**: Lighthouse score 90+ on all metrics
- **Responsive**: Mobile-first design, works on all screen sizes
- **Hosting**: Deploy to Vercel or similar for easy updates

### Notion Database Requirements
- **Prospect Database**: 
  - Properties: Name, Company, Email, Phone, Status, Consultation Date, S-1 Stack Fit, Notes, Follow-up Date
  - Views: Pipeline, By Status, Upcoming Follow-ups, All Prospects
- **Project Database**:
  - Properties: Client, Project Type, Status, Start Date, End Date, Deliverables, Team Members
  - Views: Active Projects, Completed, By Client, By Type

### Client Portal Requirements
- **Structure**: Welcome page, Project Overview, Task Tracker, Communication Hub, File Library
- **Branding**: Solvys colors, fonts, and design language
- **Automation**: Auto-create on client signup, send welcome notification
